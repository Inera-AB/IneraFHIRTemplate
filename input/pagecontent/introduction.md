# Introduktion

Denna sida beskriver informationsunderlaget som denna Implementation Guide bygger på, och ger en översiktlig bild av de arbetsflöden som förmågan ingår i. Detaljerade begrepps- och informationsmodeller, samt fullständiga processbeskrivningar, återfinns i den informationsspecifikation som anges nedan — de upprepas inte här för att undvika dubbeldokumentation som kan glida isär över tid.

### Arbetsflöde

*Ge en översiktlig bild av vilka arbetsflöden förmågan ingår i — vad som utlöser flödet och vilka aktörer som interagerar. Håll beskrivningen kort och hänvisa till informationsspecifikationen för det fullständiga, detaljerade arbetsflödet. Placera eventuella översiktsdiagram i `input/images/` och referera till dem här.*

*Exempel: "Se [informationsspecifikationen](#informationsunderlag) för den fullständiga beskrivningen av arbetsflödet. Denna IG realiserar den del av flödet som rör systemintegration via FHIR: [kort sammanfattning]."*
